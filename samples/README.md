# 샘플 엑셀 파일

이 폴더에는 테스트용 샘플 엑셀 파일이 포함되어 있습니다.

## 파일 설명

### sample-pickups.xlsx (예정)

구청에서 제공하는 형식의 샘플 엑셀 파일입니다.

**컬럼**:
- 배출위치: 수거 장소 주소
- 폐기물종류: 대형폐기물 종류
- 무게: 폐기물 무게
- 연락처: 배출자 연락처

**샘플 데이터**:
```
배출위치                     | 폐기물종류 | 무게  | 연락처
서울시 강남구 테헤란로 123    | 냉장고     | 50kg  | 010-1234-5678
서울시 서초구 서초대로 456    | 세탁기     | 40kg  | 010-2345-6789
서울시 송파구 올림픽로 300    | 침대       | 60kg  | 010-3456-7890
```

## 샘플 파일 생성 방법

1. 엑셀 프로그램 실행 (Microsoft Excel, Google Sheets 등)
2. 위의 데이터 입력
3. `sample-pickups.xlsx`로 저장
4. 웹 UI에서 업로드하여 테스트

## 주의사항

- '배출위치' 컬럼은 필수입니다
- 실제 주소가 아닌 경우 Geocoding이 정확하지 않을 수 있습니다
- 현재는 더미 좌표를 사용하므로 실제 위치와 다를 수 있습니다
