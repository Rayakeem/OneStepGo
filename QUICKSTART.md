# 빠른 시작 가이드

이 문서는 프로젝트를 빠르게 시작하기 위한 간단한 가이드입니다.

## 1. 설치

```bash
npm install
```

## 2. 빌드

```bash
npm run build
```

## 3. 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어보세요!

## 4. 사용 방법

### 엑셀 파일로 경로 최적화

1. 현재 위치 입력 (예: "서울시 중구 세종대로 110")
2. "엑셀 파일 업로드" 탭 선택
3. 엑셀 파일 선택 (샘플은 `/samples` 폴더 참고)
4. "경로 최적화 시작" 버튼 클릭

### 주소 직접 입력으로 경로 최적화

1. 현재 위치 입력
2. "주소 직접 입력" 탭 선택
3. 주소를 한 줄에 하나씩 입력:
   ```
   서울시 강남구 테헤란로 123
   서울시 서초구 서초대로 456
   서울시 송파구 올림픽로 300
   ```
4. "경로 최적화 시작" 버튼 클릭

## 5. 결과 확인

- 방문 장소 수
- 총 거리
- 예상 시간
- 최적 방문 순서

## 문제 해결

### 포트 충돌
```bash
# 다른 포트로 실행
PORT=3001 npm run dev
```

### 의존성 오류
```bash
rm -rf node_modules package-lock.json
npm install
```

더 자세한 내용은 [개발 가이드](docs/DEVELOPMENT.md)를 참고하세요.
